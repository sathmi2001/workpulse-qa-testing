# Images

## Overview

This folder contains screenshots collected during manual testing of the WorkPulse Desktop Application.

The images provide visual evidence of application behavior, reported defects, and test execution results.

## Contents

Screenshots may include:

* Application User Interface
* Validation Messages
* Functional Defects
* UI Issues
* Error Messages
* Bug Reproduction Evidence

Each screenshot corresponds to one or more documented defects in the **Bug Reports** folder and is intended to support defect analysis and verification.
